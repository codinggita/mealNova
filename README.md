## 🚀 About MealNova

MealNova is a smart tiffin and mess management platform designed for students, hostel residents, and working professionals. It allows users to customize their daily meals, manage subscriptions, and track food preferences easily.

---

## ❗ Problem Statement

Many students and working people face daily food issues:

- Same boring mess food every day  
- No option to customize meals  
- No control over diet or calories  
- Food wastage in hostels  
- Lack of proper feedback system  

MealNova solves these problems by giving full control of daily meals to users and helping vendors and hostels manage food efficiently.

## ✨ Features

### 👤 User Features
- User Signup & Login  
- Daily Meal Selection  
- Meal Customization (diet, taste, calories)  
- Subscription Management  
- Skip / Pause Meals  
- Order History  
- Feedback & Rating System  

---

### 🏢 Hostel / Mess Features
- Daily Demand Dashboard  
- Menu Management  
- Student Meal Tracking  
- Food Waste Reduction Insights  

---

### 🧑‍🍳 Vendor Features
- Add & Manage Menu  
- Accept Orders  
- Track Earnings  
- Order Management  

---

### 🔥 Smart Features
- Daily Meal Reminder Notifications  
- AI-based Meal Suggestions (based on user behavior)  
- Location-based Vendor Matching  
- Health Tracking (basic)  

---

## 🎯 Goal

To create a smart food system where users can control what they eat daily, while hostels and vendors can manage food supply efficiently and reduce waste.
